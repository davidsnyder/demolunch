h3. Installation

1. Install "MongoDB":http://www.mongodb.org/

1. Install dependencies

   @bundle@

2. Start the mongodb server

   @mongod &@

3. Start the "dl-node":https://github.com/davidsnyder/dl-node node.js vote server

4. export NODE_URL=http://localhost:3000 
   or heroku config:add NODE_URL=(http://furious-earth-8617.herokuapp.com/: + process.env.PORT) 
   
5. Start the rails server

   @rails s -p 3001@
